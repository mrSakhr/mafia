سياسة الخصوصية

آخر تحديث: 20 نوفمبر 2025

مرحبًا بك في مافيا - لعبة الأدوار الجماعية. نحن نحترم خصوصيتك ونسعى لحماية بياناتك الشخصية. توضح هذه الصفحة كيفية جمعنا واستخدامنا للبيانات داخل التطبيق.

1. البيانات التي نجمعها

أ. البيانات المستخدمة للتتبع

هذه البيانات قد تُستخدم لتتبعك عبر تطبيقات ومواقع أخرى تابعة لشركات طرف ثالث:

  • بيانات الاستخدام: تشمل التفاعل مع الإعلانات داخل التطبيق (مشاهدات الإعلانات، الضغطات).
  • الموقع التقريبي: يُستخدم لتحسين استهداف الإعلانات.
  • معرف الجهاز (Device ID / IDFA): يُستخدم لتخصيص الإعلانات وقياس أدائها.

ب. البيانات المرتبطة بك

البيانات التالية تُجمع وتُربط بمعرف جهازك:

  • معرف الجهاز (Device ID)
  • بيانات الاستخدام: التفاعل مع الإعلانات داخل التطبيق.
  • الموقع التقريبي

ج. البيانات التشخيصية

يتم جمع هذه البيانات لتحسين استقرار وأداء التطبيق. البيانات مرتبطة بمعرف الجهاز لكنها لا تُستخدم لتتبعك عبر التطبيقات أو المواقع الأخرى:

  • Crash Data: تقارير الأعطال لمساعدتنا على إصلاح المشكلات وتحسين استقرار اللعبة.
  • Performance Data: بيانات أداء التطبيق مثل زمن الإقلاع وسرعة الاستجابة لتحسين الأداء العام.

2. كيفية استخدام البيانات

  • تحسين تجربة المستخدم داخل اللعبة.
  • عرض الإعلانات المناسبة عبر AdMob.
  • إصلاح الأعطال وتحسين استقرار وأداء التطبيق.

3. مشاركة البيانات

نشارك البيانات فقط مع:

  • Google AdMob: لتحسين استهداف الإعلانات وقياس أدائها.

لا نشارك بياناتك الشخصية مع أي أطراف ثالثة أخرى.

4. أمان البيانات

نحرص على حماية بياناتك باستخدام إجراءات أمنية مناسبة لضمان سرية المعلومات وخصوصيتها.

5. معلومات إضافية

  • التطبيق لا يتطلب تسجيل دخول أو إنشاء حساب.
  • لا يتم جمع أي معلومات شخصية مباشرة مثل الاسم أو البريد الإلكتروني.

6. التواصل معنا

لأي استفسار حول الخصوصية أو جمع البيانات، يمكنك التواصل معنا عبر البريد الإلكتروني: mr.sakhr@outlook.sa
